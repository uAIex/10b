# Topic 10: NFTs and Digital Ownership Standards

CS 521 – Foundations of Crypto (Spring 2026)

This repository contains our CS 521 Topic 10 project on NFTs and digital ownership standards.

The project includes:
- A presentation on NFT standards, metadata design, marketplaces, and royalties
- An ERC-721 smart contract deployed on an Ethereum testnet
- A simple web frontend for minting and viewing NFTs

Details of the implementation and design choices will be added as the project evolves.
